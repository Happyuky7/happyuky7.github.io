# TheServerJars

A project related to Minecraft server JARs and supporting tools/utilities.

## What it is

- A web project focused on browsing/organizing server JARs and related resources.
- Built to be practical for server owners and developers.

## Highlights

- Simple, utility-first UX
- Organized data presentation
- Designed to expand with more tooling

## Tech

- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS
